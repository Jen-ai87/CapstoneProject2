# CapstoneProject2
90Mins Live Score tracking app.

## How to Run Locally

Prerequisites

Node.js v18+
npm v10+
Setup

git clone https://github.com/Jen-ai87/CapstoneProject2.git
cd 90Mins
npm install

Run Development Server

npm run dev

Build for Production

npm run build

Tech Stack

Ionic 8 + React 19
Vite 5
TypeScript 5.9
React Router 5
